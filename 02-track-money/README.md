- [MirageJS](https://miragejs.com/)

- [Styled Components](https://github.com/styled-components/styled-components)

# Execute

```bash
# Clone the repository
https://github.com/bylucaspaiva/ignite-reactjs.git
```

```bash

$ cd 02-dtmoney
```


```bash

$ yarn start
```
